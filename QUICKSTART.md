# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Настройка базы данных

1. Убедитесь, что PostgreSQL установлен и запущен
2. Создайте базу данных:

```bash
createdb flower_shop
```

Или через psql:

```sql
CREATE DATABASE flower_shop;
```

3. Выполните SQL скрипт:

```bash
psql -U postgres -d flower_shop -f database.sql
```

## Шаг 3: Настройка переменных окружения

Создайте файл `.env`:

```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите ваши параметры подключения к БД.

## Шаг 4: Запуск приложения

```bash
npm start
```

Или для разработки с автоперезагрузкой:

```bash
npm run dev
```

## Шаг 5: Откройте в браузере

Перейдите по адресу: http://localhost:3000

## Готово!

Теперь вы можете:
- Создавать, редактировать и удалять заказы
- Добавлять позиции в заказы
- Перемещать позиции между заказами
- Переключать системную дату
